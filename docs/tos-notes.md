# ToS Notes

This is not legal advice.

Goodreads does not provide a current public API for new integrations. This project maps user-visible web surfaces and public feeds for personal backup, migration, and automation. Keep behavior conservative:

- conservative read/write behavior;
- user-owned account context only;
- no credential sharing;
- no bypassing paywalls or access controls;
- no high-rate crawling;
- live writes should be intentional, account-owned, and verified after execution.
